Detailed Description

Controls the reading and compilation of a COBOL file, start to finish.

When constructed, a Driver identifies the location of each COBOL division present in the file. To compile, the Compile function then calls its own handler function for each division, which lexes, parses, generates code for, and optimizes for that division.
